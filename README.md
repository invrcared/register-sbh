## How to Register

Registration is handled entirely through Pull Requests. This "Infrastructure as Code" approach ensures that all records are public, transparent, and easy to manage.

- [Fork](https://github.com/is-a-site/register/fork) the repository.
- Follow the instructions on our [documentation](https://docs.is-a.site).
- Once you open your pull request (PR), it will be reviewed. *Keep an eye on it in case changes are needed!*
   - If changes have been requested, please make the specified changes otherwise **you will be rejected**.
- Once your PR is merged, your DNS records should be published with-in a few minutes.
- Enjoy your new `.is-a.site` subdomain! Please consider leaving a star ⭐️ to help support us!

## Report Abuse
If you find any subdomains being abused or breaking our TOS, please report them by [creating an issue](https://github.com/is-a-site/register/issues/new?assignees=&labels=report-abuse&projects=&template=report-abuse.md&title=Report+abuse) with relevant evidence.
